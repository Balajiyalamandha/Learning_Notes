# ‚úÖ Case 2: Run Jenkins Agent on another Windows Machine

If you want another Windows system to run builds (agent):

### Step 1: On second Windows machine

* Install Java
* Create folder:

  ```
  C:\Jenkins\workspace
  ```

### Step 2: In Jenkins Master (192.168.0.84)

* Go to:
  **Manage Jenkins ‚Üí Manage Nodes and Clouds ‚Üí New Node**
* Create node (e.g., `Windows-Agent2`)
* Set:

  * Remote root directory ‚Üí `C:\Jenkins\workspace`
  * Launch method ‚Üí Launch agent via WebSocket

### Step 3: On second machine, run:

```bash
java -jar agent.jar -url http://192.168.0.84:8080/ -secret <secret> -name Windows-Agent2 -webSocket -workDir "C:\Jenkins\workspace"
```

If connected successfully, you will see:

```
WebSocket connection open
Connected
```
